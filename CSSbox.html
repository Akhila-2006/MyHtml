CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the
 presentation of a webpage written in HTML. It controls the layout, colors, fonts, and 
 overall design of web pages, ensuring they look visually appealing and function 
 well on different devices.
 
    
Common CSS Styles:
Text Styling: color, font-size, text-align
    Box Model: margin, padding, border
    Layout: display, position, flexbox, grid 
    Background: background-color, background-image
    Animations: transition, animation
