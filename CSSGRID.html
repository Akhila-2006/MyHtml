CSS
CSS is the language used to style and design HTML elements. It controls colors, fonts, layout, and overall look of the webpage.
HTML
HTML is the standard language used to create and structure webpages using elements like headings, paragraphs, links, images, and more.
JavaScript
JavaScript is a programming language used to add interactivity and logic to websites. It makes webpages dynamic and responsive to user actions.
